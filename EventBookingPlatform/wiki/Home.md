# Event Booking Platform Wiki

## Project summary
Event Booking Platform is a web-based ASP.NET Core MVC application for managing events and selling tickets online.

## Architecture
- Presentation Layer: Controllers + Razor Views
- Business Layer: Services + Interfaces
- Data Layer: EF Core + SQLite
- DTO Layer: Safe data transfer between application layers

## Implemented requirements
- Data persistence with EF Core
- Minimum three related tables
- LINQ queries
- BaseEntity inheritance
- Interfaces and polymorphism
- MVC architecture
- Dependency Injection
- DTOs
- CRUD operations
- Authentication and authorization
- Data validation
- Bootstrap UI
- Statistics and reports

## Future improvements
- QR code tickets
- Email confirmations
- Payment gateway
- Export reports to PDF/Excel
- Dashboard charts
